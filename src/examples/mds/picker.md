# Picker选择器

```
##:Picker:##
```

#### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| value     | 值   | String  |   -       |    -    |
| placeholder     | 占位文本   | String  |   -       |    请选择    |
| options     | 选项列表   | Array[{value,label}]  |   -       |    -    |

#### Events
| 事件名称 | 说明 | 回调参数 |
|---------|--------|---------|
| change | 值改变触发事件 | value |
