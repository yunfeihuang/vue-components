# Range滑块输入框

```
##:Range:##
```

#### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| -     | -   | -  |   -       |    -    |
| value     | 值   | Number  |     -     |    -    |
| max     | 最大值   | Number  |     -     |    100    |
| min     | 最小值   | Number  |     -     |    0    |
| step     | 步值   | Number  |     -     |    1    |
| disabled     | 是否禁用   | Boolean  |   -       |    false    |
| unit     | 单位   | String,Function  |     -     |    -    |

#### Events
| 事件名称 | 说明 | 回调参数 |
|---------|--------|---------|
| change | - | value |
