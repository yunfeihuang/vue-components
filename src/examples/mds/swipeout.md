# Swipeout操作

```
##:Swipeout:##
```

#### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| disabled     | 是否禁用   | Boolean  |   -       |    false    |
| open     | 是否打开操作项   | Boolean  |   -       |    false    |
| divider     | 是否底分割线   | Boolean  |   -       |    false    |
| open     | -   | -  |   -       |    -    |

#### Events
| 事件名称 | 说明 | 回调参数 |
|---------|--------|---------|
| open | 打开事件 | - |
| close | 关闭事件 | - |

#### Slots
| 名称 | 说明 | 
|---------|--------|
| default | - |
| action | 操作项 |
