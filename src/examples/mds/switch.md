# Switch开关

```
##:Switch:##
```
#### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| -     | -   | -  |   -       |    -    |
| value     | 值   | String,Boolean  |     -     |    -    |
| disabled     | 是否禁用   | Boolean  |   -       |    false    |
| size     | 是否小风格样式   | String  |   default,small       |    default    |
| onValue     | 选中的value设定值   | String  |   -       |    -    |
| offValue     | 没有选中的value设定值   | String  |   -       |    -    |

#### Events
